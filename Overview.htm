Tool : JavaBean and EntityBean Generator

Description

This tool will be used to generate a javabean from a data source. The data source can be a RDBMS database table or a XSD schema. Tool will read the schema from table or XSD and will generate a bean source code.


Design steps

Step 1: Connect to a datasource

  • Input : JDBC Datasource* credentials with jndi name OR url to XSD schema**
  • Output : A connection to Datasource or url connection to XSD schema

Step 2: Retrieve database table/XSD schema columns

  • Input : Datasource connection and name of the table OR XSD file content
  • Output : Java object with table columns/xsd schema definition populated

Step 3: Generte a JavaBean or Entity Bean source code

  • Input : Source file detail (Name, path, source template api interface etc.)
  • Output : JavaBean/Entity Bean source


*Datasource

We will use JDBC datasource (JDBC API 3.0) as a preferred method to lookup and connect to a datasource
For More information refer http://java.sun.com/j2se/1.4.2/docs/guide/jdbc/getstart/datasource.html
